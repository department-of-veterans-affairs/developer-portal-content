We added these new endpoints to version 1 of the Clinical Health API:
- GET /Appointment
- POST /DocumentReference

These resources expand clinician-facing functionality and support improved Veteran care. 

**GET /Appointment**

This endpoint returns information about a clinician’s past or upcoming appointments from the VA’s Electronic Health Record (EHR). 

**POST /DocumentReference**

This endpoint allows clinicians to add notes to a patient’s chart in VA’s EHR. 

Access to these endpoints is currently restricted to consumers who meet the appropriate use cases and security requirements. All access requests will be carefully reviewed.

To learn more, visit the [Clinical Health API documentation](https://developer.va.gov/explore/api/clinical-health/docs?version=current).
