We've added the following 2 new endpoints to version 1 of the Clinical Health API:

 Read Clinician Appointment Data: GET https://api.va.gov/services/clinical-fhir/v1/r4/Appointment?
 
 Write Clinical Notes: POST https://api.va.gov/services/clinical-fhir/v1/r4/DocumentReference/


Read Clinician Appointment Data provides the ability to read clinician appointments from VA's EHR and Write Clinical Notes enables clinicians to write-back notes related to patient appointments in VA's EHR. Both of these resources expand the existing clinician-facing functionality to enhance veteran care.

Access to these APIs is currently restricted to consumes who meet the appropriate use cases and security requirements. Requests for using these resources will require thorough vetting before access is granted.
