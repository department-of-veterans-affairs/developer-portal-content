**Version 0 of the Clinical Health API is now deprecated.** 

We won’t add new features or provide ongoing maintenance for this version. We’ll continue to honor service-level agreements and consider critical defects, but we discourage using version 0. 

We’re deprecating this version because we’re migrating from a legacy single sign-on (SSO) technology to Microsoft Entra ID, a modern SSO platform. Version 1 supports Entra ID authorization and will be the only version we maintain going forward. 

Both Authorization Code Grant (ACG) and Client Credentials Grant (CCG) consumers should move to version 1.

**Version 0 will be deactivated in April 2026.**

After deactivation, it will be removed from the developer portal and can’t be used.
