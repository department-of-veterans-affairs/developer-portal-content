Version 0 of the Clinical Health API is now deprecated.


When we deprecate a version of an API, it will still be supported, but its use is discouraged and it will be deactivated at a future date. We’ll honor service-level agreements, but no new features will be added to the API version and only critical defects will be considered for correction.

We've deprecated this API version because Authorization Client Grant (ACG) consumers using single sign-on internal (SSOi) must migrate to Microsoft Entra ID by April 1, 2026. Version 1 of the API supports Entra ID authorization and will be the only version maintained going forward, therefore both ACG and Client Credentials Grant (CCG) consumers should move to Version 1
